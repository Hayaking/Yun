## 运

pyqt5 + python3.6(anaconda) + eric6  

执行action.py即可运行(run action.py)

|吸星大法|吸取对手全部气，可被对手挡、小波抵消掉(不造成伤害)，如果此时对手使用运，则不产生任何效果|5气|
